# SPACES

A conceptual art worm that changes the name of all your 
files and folders to whitespace. 

#### USE AT YOUR OWN RISK! This code will wipe EVERYTHING on your drive 🤖😭😈🤦‍♂️

This...

    folder/
     ⌞ file1.txt
     ⌞ file2.txt
     
    anotherFolder/
     ⌞ file3.wav
     ⌞ file4.mp4

...will become this

    '      '/
     ⌞ '     '.'   '
     ⌞ '     '.'   '
     
    '             '/
     ⌞ '     '.'   '
     ⌞ '     '.'   '

(No quotes, of course!)
